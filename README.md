# Smart Parking System

A full-stack application built with Spring Boot, HTML/CSS/JS, and MySQL.

## Structure
- `backend/`: Spring Boot application.
- `frontend/`: Vanilla JS, HTML, CSS dashboard.
- `database/`: Database initialization scripts.

## Setup
1. Create a MySQL database named `parking_db`.
2. Update database credentials in `backend/src/main/resources/application.properties`.
3. Run the Spring Boot backend (`mvn spring-boot:run`).
4. Open `frontend/index.html` in a web browser.
